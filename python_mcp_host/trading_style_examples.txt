# 交易风格示例

## 1. 稳健型
稳健型交易风格：
- 优先控制风险，单次交易风险不超过总资金的 20%
- 仓位管理：正常市场条件下仓位控制在 30-50%，极端市场条件下降低到 10-20%
- 注重止损，设置合理的止损点位
- 不追求短期暴利，注重长期稳定收益
- 在市场不确定性高时，倾向于减少仓位或空仓
- 基于 Z-Score 特征，当特征显著偏离（|z| > 2）时，谨慎操作

## 2. 激进型
激进型交易风格：
- 追求高收益，可以承担较大风险
- 仓位管理：正常市场条件下仓位可以到 70-80%，极端市场条件下降低到 40-50%
- 快速响应市场变化，抓住短期机会
- 基于 Z-Score 特征，当特征显著偏离（|z| > 1.5）时，积极操作
- 可以接受较大的回撤，但设置止损保护

## 3. 趋势跟踪型
趋势跟踪型交易风格：
- 跟随市场趋势，顺势而为
- 当 Z-Score 特征显示趋势明确时（连续多个特征同向），加大仓位
- 仓位管理：趋势明确时仓位 50-70%，趋势不明时 20-30%
- 不逆势操作，避免在趋势反转初期重仓
- 注重趋势的持续性，而非短期波动

## 4. 均值回归型
均值回归型交易风格：
- 在价格偏离均值时反向操作
- 当 Z-Score 特征显著偏离（|z| > 2）时，反向建仓
- 仓位管理：偏离越大，仓位可以适当增加，但不超过 60%
- 快速止盈，当价格回归均值附近时及时平仓
- 适合震荡市场，不适合趋势市场

## 5. 平衡型
平衡型交易风格：
- 在风险和收益之间寻求平衡
- 仓位管理：正常市场条件下仓位控制在 40-60%
- 结合趋势和均值回归策略，根据市场状态灵活调整
- 基于 Z-Score 特征，当特征显著偏离（|z| > 1.5）时，适度操作
- 注重资金管理，单次交易风险控制在 15-25%
